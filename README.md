# Task-Manegement-App

タスクの登録、更新、削除ができるアプリ。Reactの練習用に作成。

## 使用技術
- JavaScript
  - React
  - TypeScript
- CSS
  - Sass
  - Tailwind CSS（一部）
- ビルドツール
  - Vite
- ライブラリ
  - react-beautiful-dnd
